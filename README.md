# 🖼️ Image Caption Generator using BLIP

Ever wondered how machines can *look at an image and describe it in words*?
This project does exactly that.

It’s an AI-powered web app that takes an image as input and generates a meaningful caption using the BLIP (Bootstrapping Language-Image Pretraining) model.

---

## 🚀 What this project does

* Upload any image and get an instant caption
* Understands objects and scenes using deep learning
* Provides a clean and interactive interface
* Runs efficiently using pre-trained models

---

## 🧠 Tech behind the scenes

This project combines computer vision and natural language processing:

* Python
* PyTorch
* Hugging Face Transformers
* Gradio (for UI)
* PIL for image processing

---

## 📸 How it works (simple flow)

1. You upload an image
2. The BLIP model processes visual features
3. The model generates a caption
4. The caption is displayed instantly

---

## 💻 Getting started

Clone this repository:

```bash
git clone https://github.com/Navneet1135/blip-image-caption-generator.git
cd blip-image-caption-generator
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the app

```bash
python app.py
```

Once you run this, a local web interface will open where you can upload images and test the model.
